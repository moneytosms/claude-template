# Plugins

Plugins are installed at onboarding (not vendored here), e.g. via:

```
/plugin marketplace add <owner/repo>
/plugin install <name>@<marketplace>
```

The onboarding skill installs the default set listed in
[`.claude/install-manifest.md`](../install-manifest.md) and any stack-specific
plugins it recommends. Record what you installed here for the team.
